# Guia do Operador de Balcao

## Como abrir o sistema

1. Abra o navegador.
2. Acesse o link informado pelo responsavel.
3. Deixe o leitor NFC conectado ao computador.

## Como cadastrar cliente

1. Clique em `Novo cliente`.
2. Preencha nome.
3. Preencha telefone se tiver autorizacao.
4. Use CPF apenas se houver autorizacao.
5. Clique em cadastrar.

## Como vincular tag NFC

1. Clique em `Vincular NFC`.
2. Clique no campo UID.
3. Aproxime a tag NFC do leitor.
4. Selecione o cliente.
5. Clique em vincular.

## Como consultar cliente por NFC

1. Clique em `Consultar NFC`.
2. Aproxime a tag NFC do leitor.
3. Aguarde o nome do cliente aparecer.
4. Clique em `Limpar e ler outra tag` para atender outra pessoa.

## Se a tag nao aparecer

1. Clique novamente no campo UID.
2. Aproxime a tag mais devagar.
3. Teste a tag no Bloco de Notas.
4. Chame o responsavel tecnico se continuar falhando.

## Se aparecer "Tag NFC nao encontrada"

1. Verifique se a tag ja foi vinculada.
2. Abra `Vincular NFC`.
3. Vincule a tag ao cliente correto.

## Se a internet cair

1. Pare o uso do sistema.
2. Registre o atendimento manualmente.
3. Avise o responsavel.
4. Volte a usar quando a internet retornar.

## O que nao fazer

- Nao cadastrar dados reais sem autorizacao.
- Nao deixar o sistema aberto sem supervisao.
- Nao compartilhar o link com pessoas fora do piloto.
- Nao anotar CPF ou telefone em locais inseguros.
