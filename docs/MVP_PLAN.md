# Plano do MVP

## Objetivo

Criar a base estrutural de um sistema NFC para pequenas cidades, permitindo validar rapidamente se o fluxo de leitura, cadastro, vinculo e consulta por UID resolve o problema principal.

## Escopo Inicial

- Ler UID de tag NFC.
- Cadastrar cliente.
- Vincular UID NFC ao cliente.
- Consultar cliente pelo UID.
- Preparar base para descontos e historico futuramente.

## Fora do Escopo Inicial

- Autenticacao.
- Telas completas.
- Banco real.
- Regras de desconto.
- Historico funcional.
- Integracao definitiva com leitor NFC via biblioteca.
