## Clientes

## Tags NFC

- `POST /nfc-tags`: cadastrar UID NFC.
- `POST /nfc-tags/vincular`: vincular UID NFC a um cliente.
- `GET /nfc-tags/{uid}`: consultar tag NFC e cliente vinculado por UID.
