# Metricas do Piloto

## Metricas Principais

- Tempo medio de cadastro.
- Tempo medio de vinculo NFC.
- Tempo medio de consulta NFC.
- Quantidade de clientes cadastrados.
- Quantidade de tags vinculadas.
- Quantidade de consultas realizadas.
- Quantidade de falhas.
- Quantidade de tags nao encontradas.
- Duvidas do operador.
- Cliques necessarios por tarefa.

## Metricas de Decisao

- O operador conseguiu usar sozinho?
- O cliente entendeu o uso da tag?
- A leitura foi rapida?
- A tela de consulta foi clara?
- O sistema travou?
- O fluxo faz sentido no balcao?
